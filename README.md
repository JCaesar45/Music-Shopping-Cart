# 🎵 Music Shopping Cart

A simple shopping cart web page built with **HTML** and **Tailwind CSS**.  
This project is designed to showcase responsive layouts, Flexbox utilities, and reusable card components using Tailwind’s utility classes.

---

## 🚀 Features

- Responsive Flexbox layout (`flex-col` on mobile, `lg:flex-row` on large screens)
- Product cards with:
  - Product name
  - Quantity
  - Price
  - Remove button (with hover state)
- Order summary container with:
  - Rounded corners and border styling
  - Order total and amount
  - Checkout button (hover effects included)

---

## 📂 Project Structure

``

music-shopping-cart/
├── index.html        # Main page
├── README.md         # Project documentation

``

---

## 🛠️ Technologies Used

- **HTML5**
- **Tailwind CSS** (utility-first CSS framework)

---

## 📋 User Stories Completed

- ✅ `#shopping-cart-container` with Flexbox layout  
- ✅ Responsive column → row layout using `flex-col` and `lg:flex-row`  
- ✅ `#products-container` with multiple `.card` elements  
- ✅ Product card elements:  
  - `h2` with Tailwind text size class  
  - `.quantity` element  
  - `.price` element  
  - `.remove-button` with `bg-red-*` and `hover:bg-red-*` states  
- ✅ `#order-summary-container` with:  
  - Rounded corners (`rounded-lg`)  
  - Border (`border`)  
  - `h2` heading (`text-2xl`)  
  - `#total` element with bold text and font size  
  - `#total-amount` for displaying cart total  
  - Checkout link with `bg-blue-*` and hover state
